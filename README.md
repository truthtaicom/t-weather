# Demo Weather App

**Technologies**:
- [x] TypeScript
- [x] ReactJS
- [x] Redux
- [x] Redux Thunk
- [x] Storybook
- [x] Styled-Component
- [x] Jest
- [x] Cypress


**Scripts**
- Install the application: `npm install`
- Run Application: `npm run start`
- Run Storybook: `npm run storybook`
- Run Jest test ( Unit-testing): `npm run test`
- Run Cypress ( End-to-end testing): `npm run cypress`
